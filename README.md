# Central de Tarefas - Ciencia de Dados (EBAC)

Este repositório funciona como um indice unico para organizar todas as tarefas do curso de Ciencia de Dados da EBAC.

Objetivo:
- Facilitar a avaliacao dos professores.
- Ajudar voce a encontrar rapidamente qualquer modulo.
- Manter um padrao unico de organizacao ao longo do curso.

## Como adicionar um novo modulo

1. Crie o repositório da tarefa no seu GitHub.
2. Adicione uma nova linha na tabela acima com:
	- Nome do modulo
	- Status (Concluido, Em andamento, Nao iniciado)
	- Link do repositório
	- Observacoes (opcional)
3. Atualize o campo Total de modulos com repositorio registrado.
4. Atualize a data em Ultima atualizacao.

## Modelo de linha para copiar

Use este modelo na tabela:

| Modulo XX | Em andamento | [nome-do-repo](https://github.com/seu-usuario/nome-do-repo) | Descricao curta |

## Convencao recomendada para nomes de repositorios

Padrao sugerido:

`data-science-modXX`

Exemplos:
- data-science-mod15
- data-science-mod16

Assim, os repositórios ficam consistentes e faceis de localizar.
